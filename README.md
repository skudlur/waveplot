# waveplot
waveplot is a VCD waveform generator for the terminal.

### Requirements
- cargo (Rust)

### Installation
```bash
cargo install waveplot
```

### Usage 
- help
```bash
waveplot --help / -h
```

- version
```bash
waveplot --version / -V
```

- Generate a waveform from a VCD file
```bash
waveplot <path_to_vcd_file> 
```
